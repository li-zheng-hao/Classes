# 捕鱼达人
### -已经完成的功能：</br>
1、加载界面：资源的异步加载、资源加载进度条的实现</br>
2、游戏主界面：炮台旋转角度的实现、炮台等级的切换、炮台子弹与鱼类之间的碰撞检测、鱼类的游动 </br>
